### installation

```shell
cd ~ && git clone https://github.com/ozh/dotfiles-raspi && cd dotfiles-raspi && source ~/dotfiles-raspi/install.sh
```

### update

```shell
cd ~/dotfiles-raspi && git pull && source ~/dotfiles-raspi/install.sh
```
